# 热点地图测试指南

## 🧪 快速测试清单

打开 `hotspot-map-prototype.html` 后，按照以下步骤测试：

### 1️⃣ 基础功能测试

- [ ] 页面正常加载，无错误提示
- [ ] 地图显示正常
- [ ] 左侧侧边栏显示完整
- [ ] 右下角显示影响力图例

### 2️⃣ 热点展示测试

- [ ] 地图上显示 12 个热点
- [ ] 热点圈有脉冲动画效果
- [ ] 不同影响力热点显示不同颜色
- [ ] 点击热点圈可以打开 Popup

### 3️⃣ Popup 详情测试

- [ ] Popup 显示热点标题和平台图标
- [ ] Popup 显示影响力分数和等级
- [ ] Popup 显示热点类型标签
- [ ] Popup 显示详细数据（点赞、收藏等）
- [ ] 点击地图空白处 Popup 关闭

### 4️⃣ 筛选功能测试

**平台筛选：**
- [ ] 取消勾选「小红书」，地图上只显示微博、抖音、B站热点
- [ ] 重新勾选「小红书」，小红书热点重新显示
- [ ] 取消勾选所有平台，热点全部消失
- [ ] 重新勾选某个平台，该平台热点重新显示

**类型筛选：**
- [ ] 取消勾选「美食」，只保留其他类型热点
- [ ] 只勾选「ACG」，只显示 B站的 3 个热点
- [ ] 同时勾选多个类型，显示符合条件的热点

### 5️⃣ 地图切换测试

- [ ] 点击「🌍 世界地图」
- [ ] 地图视角切换到世界地图
- [ ] 点击「🇨🇳 中国地图」
- [ ] 地图视角切换回中国地图

### 6️⃣ 统计功能测试

- [ ] 侧边栏显示正确的热点总数（初始 12）
- [ ] 各平台统计数量正确
- [ ] 筛选后统计数据实时更新

### 7️⃣ 地图操作测试

- [ ] 鼠标滚轮可以缩放地图
- [ ] 鼠标拖拽可以移动地图
- [ ] 缩放后热点圈大小保持合理

## 🐛 常见问题

### Q1: 地图无法加载
**可能原因**：网络问题，地图瓦片无法下载

**解决方法**：
- 检查网络连接
- 尝试切换网络（手机热点）
- 如果在中国，使用中国地图模式（默认）

### Q2: 热点不显示
**可能原因**：筛选条件过于严格

**解决方法**：
- 检查平台筛选是否全部取消
- 检查类型筛选是否全部取消
- 重新勾选至少一个平台和类型

### Q3: 动画效果不明显
**可能原因**：浏览器性能问题

**解决方法**：
- 尝试使用 Chrome 浏览器
- 减少打开的标签页
- 关闭其他占用资源的程序

### Q4: Popup 内容显示不全
**可能原因**：浏览器缩放问题

**解决方法**：
- 按下 Cmd+0（Mac）或 Ctrl+0（Windows）恢复默认缩放
- 拖动 Popup 到合适位置
