# Weighted Random Distribution & Pseudo-Random Systems

## 核心概念

### 1. 加权随机 (Weighted Random)
加权随机是一种概率分配机制，使某些结果比其他结果更有可能出现。每个选项都有一个权重值，权重越高，被选中的概率越大。

#### 数学原理
```
P(item_i) = weight_i / Σ(weight_j for all j)
```

其中：
- P(item_i) = 选中第i个选项的概率
- weight_i = 第i个选项的权重
- Σ(weight_j) = 所有选项的权重总和

#### 应用价值
- **资源优化**: 精准分配有限资源给高价值目标
- **公平性平衡**: 既保证整体公平，又体现差异化
- **用户体验**: 为不同用户提供个性化体验
- **成本控制**: 在预算约束下最大化收益

---

### 2. 伪随机分布 (Pseudo-Random Distribution)
伪随机是通过确定性算法生成的看似随机的数列。给定相同的种子，会产生相同的"随机"序列。

#### 关键特性
- **可重现性**: 相同种子 → 相同结果
- **可预测性**: 掌握算法和种子即可预测
- **可控性**: 通过调整种子影响结果分布

#### 在商业中的应用
1. **用户体验一致性**: 同一用户在不同时间看到相同内容
2. **A/B测试**: 控制变量，确保测试有效性
3. **审计追踪**: 可重现的结果便于问题排查
4. **公平性验证**: 避免真正的随机带来的争议

---

### 3. 组合应用：加权伪随机系统

#### 核心算法
```
1. 计算每个选项的基础权重
2. 根据上下文因子调整权重
3. 使用确定性种子生成伪随机数
4. 基于调整后的权重进行加权随机选择
5. 输出确定性的结果
```

#### 优势
- ✅ 个性化 + 可预测
- ✅ 资源优化 + 成本控制
- ✅ 用户体验 + 商业目标
- ✅ 可审计 + 可优化

## 数学基础

### 权重归一化
```python
def normalize_weights(weights):
    total = sum(weights)
    return [w / total for w in weights]
```

### 累积分布函数
```python
def cumulative_distribution(weights):
    """构建CDF用于快速查找"""
    return [sum(weights[:i+1]) for i in range(len(weights))]
```

### 二分查找优化
```python
import bisect

def weighted_choice(items, weights):
    """使用二分查找优化，O(log n)"""
    cdf = cumulative_distribution(weights)
    r = random.random()
    idx = bisect.bisect_left(cdf, r)
    return items[idx]
```

## 常见陷阱

### ❌ 避免做法
1. **权重过大差异**: 10000 vs 1 导致其他选项几乎不可能选中
2. **种子泄露**: 用户可预测并利用随机种子
3. **权重硬编码**: 无法根据业务数据动态调整
4. **忽略边界情况**: 权重全为0、负权重等

### ✅ 最佳实践
1. **权重平滑**: 对数变换防止极端差异
2. **种子加密**: 使用用户ID + 时间戳的哈希值
3. **数据驱动**: 基于历史数据持续优化权重
4. **异常处理**: 优雅降级，保证系统稳定性

## 高级优化

### 1. 自适应权重

### 3. 冷启动处理
