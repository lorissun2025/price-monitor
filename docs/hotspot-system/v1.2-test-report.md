## 🐛 已知问题和限制

### 1. 浏览器兼容性
- **问题：** 部分旧版浏览器可能不支持 Geolocation API
- **影响：** Safari 10 以下、IE 11 及以下
- **解决：** 建议使用现代浏览器（Chrome 90+、Firefox 88+、Safari 15+）

### 2. 定位精度
- **问题：** 使用网络定位，精度可能较低（100-1000 米）
- **影响：** 距离计算可能有误差
- **解决：** 这是 Geolocation API 的限制，无法完全解决

### 3. 隐私模式
- **问题：** 浏览器隐私模式可能禁用 Geolocation
- **影响：** 无法获取用户位置
- **解决：** 建议用户退出隐私模式

### 4. HTTPS 限制
- **问题：** Chrome 等浏览器要求 HTTPS 才能使用 Geolocation（localhost 除外）
- **影响：** 部署到 HTTP 网站时无法定位
- **解决：** 部署时使用 HTTPS
