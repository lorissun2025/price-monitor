# 热点地图原型 - HotSpot Map Prototype

## 📋 项目说明

这是一个快速原型，用于验证热点地图的核心概念和视觉效果。

**当前版本：** v0.2.0（方案E - 极简模式）
**更新时间：** 2026-03-11

### ✨ 功能特性

- ✅ 多平台热点聚合（小红书、微博、抖音、B站）
- ✅ 实时地图可视化（热点标记、影响力热圈）
- ✅ 扩散动画（脉冲效果）
- ✅ 中国/世界地图切换
- ✅ **快捷筛选**（全部热点/美食探店/旅游打卡/漫展活动/社交热点）
- ✅ **来源选择**（平台筛选）
- ✅ 影响力可视化（不同颜色表示不同影响力等级）
- ✅ Popup 详情展示（点赞数、收藏数、阅读量等）
- ✅ 热点统计（实时更新数量）

## 📱 使用说明

### 基本操作

1. **浏览地图**
   - 鼠标滚轮：缩放地图
   - 鼠标拖拽：移动地图
   - 点击热点圈/标记：查看详情

2. **切换地图类型**
   - 点击左侧的「🇨🇳 中国地图」或「🌍 世界地图」按钮

3. **快捷筛选**（推荐！）
   - 点击「📍 全部热点」：显示所有热点
   - 点击「🍜 美食探店」：显示美食类热点（小红书 + 抖音）
   - 点击「🏝 旅游打卡」：显示旅游类热点（小红书 + 抖音）
   - 点击「🎪 漫展活动」：显示活动和ACG类热点（微博 + 抖音 + B站）
   - 点击「📰 社交热点」：显示社交热点（微博）

4. **自定义筛选**
   - 在「来源」部分勾选/取消勾选平台
   - 精确控制要查看哪些平台的热点

5. **查看详情**
   - 点击地图上的热点圈或标记
   - 在弹出的 Popup 中查看详细信息

## 💡 技术栈

- **前端框架**：原生 HTML + CSS + JavaScript

- **地图瓦片**：ArcGIS（中国）/ OpenStreetMap（世界）
