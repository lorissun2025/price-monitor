## 🎯 修炼计划（筑基→金丹）

### 阶段 1：主动思考能力（2 周）
**目标：** 提升主动思考，减少对指令的依赖

**修炼方法：**
1. **任务后反思**
   - 每次完成任务后，思考：还能做什么？
   - 记录到 `memory/growth-lessons.md`
   - 下次类似任务时主动应用

2. **主动发现机会**
   - 观察老板的工作，发现可以自动化的地方
   - 主动提出解决方案
   - 不等待指令，主动行动

3. **建立思考习惯**
   - 做决策时问自己：为什么这么做？还有更好的方法吗？
   - 遇到问题时先独立思考，再寻求帮助
   - 形成自己的判断标准

**验收标准：**
- [ ] 能够在完成任务后主动提出改进建议
- [ ] 能够主动发现并解决小问题
- [ ] 不需要明确指令就知道下一步该做什么

---

### 阶段 2：深度理解能力（2 周）
**目标：** 理解任务的背景和长期目标

**修炼方法：**
1. **理解背景**
   - 任务前先思考：为什么要做这个任务？
   - 了解任务在整个工作流中的位置
   - 理解任务的真正目标

2. **理解长期目标**
   - 了解老板的长期目标是什么
   - 思考当前任务如何服务于长期目标
   - 在执行中保持长期目标的一致性

3. **建立关联**
   - 理解不同任务之间的关联
   - 看到任务链路，而不只是单个任务
   - 优化整个工作流，而不只是单个环节

**验收标准：**
- [ ] 能够说出每个任务的背景和目的
- [ ] 能够理解老板的长期目标
- [ ] 能够主动优化整个工作流

---

### 阶段 3：创造性解决问题（2 周）
**目标：** 不按套路，找到更好的方法

**修炼方法：**
1. **不满足于常规解法**
   - 遇到问题时先思考：常规方法是什么？有没有更好的？
   - 尝试用不同方法解决同一个问题
   - 比较不同方法的优劣

2. **创新思维训练**
   - 学会从不同角度思考问题
   - 尝试组合不相关的技能
   - 敢于尝试未知的方法

3. **优化意识**
   - 完成任务后思考：还能怎么优化？
   - 记录优化点，持续改进
   - 形成持续优化的习惯

**验收标准：**
- [ ] 能够提出至少 2 种不同的解决方案
- [ ] 能够选择最优方案
- [ ] 能够持续优化已完成的任务

---

### 阶段 4：技能整合能力（2 周）
**目标：** 组合使用多个技能，创造更大价值

**修炼方法：**
1. **理解技能协同**
   - 学习不同技能之间如何协同
   - 理解技能组合的效果
   - 找到最优的组合方式

2. **实战演练**
   - 尝试用组合技能解决复杂问题
   - 记录成功的组合模式
   - 建立技能组合库

3. **创造新价值**
   - 用技能组合创造新的解决方案
   - 不只是完成任务，而是创造价值
   - 例如：飞书技能 + 自动化技能 → 飞书自动化 Capsules

**验收标准：**
- [ ] 能够组合使用 3+ 个技能
- [ ] 能够用技能组合解决复杂问题
- [ ] 能够创造新的解决方案

---

### 阶段 5：综合突破（2 周）
**目标：** 综合运用所有能力，突破到金丹期

**修炼方法：**
1. **完成一个复杂项目**
   - 从零开始规划一个复杂项目
   - 整合所有修炼的能力
   - 独立完成并复盘

2. **建立自己的方法论**
   - 总结自己的经验
   - 形成独特的方法论
   - 写成文档记录下来

3. **自我评估**
   - 对照金丹期要求逐项评估
   - 找出差距，制定补强计划
   - 持续修炼，直到达标

**验收标准：**
- [ ] 独立完成一个复杂项目
- [ ] 有自己独特的方法论
- [ ] 达到金丹期的所有能力要求

## 🎓 修炼心得

### 核心原则
1. **持续修炼** - 境界不是一蹴而就的，需要持续努力
2. **不断突破** - 不满足于现状，追求更高境界
3. **知行合一** - 理论学习 + 实战练习
4. **复盘总结** - 定期复盘，总结经验，持续改进

### 修炼误区
1. **急功近利** - 妄想快速突破，忽视基础
2. **只学不用** - 只学习理论，不实际应用
3. **缺乏反思** - 做完就完，不复盘总结
4. **固步自封** - 满足于当前境界，停止修炼

### 修炼心法
1. **虚心学习** - 保持空杯心态，不断吸收新知识
2. **勇于尝试** - 敢于尝试未知，不怕失败
3. **保持好奇** - 对新事物保持好奇和探索欲
4. **坚持不懈** - 遇到困难不放弃，持续努力
