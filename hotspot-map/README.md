# 🗺️ 热点地图 - HotSpot Map

> 地理位置化的热点内容可视化平台

## 📋 项目简介

热点地图系统是一个地理位置化的热点内容可视化平台，聚合多个主流社交媒体平台的热点内容，通过地图形式直观展示热点分布、影响力、趋势变化。

**版本:** v1.2

## ✨ 核心功能

- 📍 **多平台数据聚合** - 小红书、微博、抖音、B站
- 🗺️ **地图可视化展示** - 中国地图、世界地图
- 🔍 **智能搜索功能** - 按标题、城市、作者搜索
- 🎯 **智能筛选系统** - 类型筛选、平台筛选
- 🔄 **实时数据刷新** - 一键获取最新热点
- 📊 **影响力评分** - 多维度分析热点传播度
- 📈 **实时统计面板** - 热点数量、分布统计
- 🚀 **用户位置定位** - 一键定位，查看附近热点
- 📏 **距离计算筛选** - 按距离筛选景点，显示距离信息
- 🗺️ **距离排序功能** - 按距离从近到远排序
- 📊 **距离统计面板** - 显示最近、平均距离及分布

## 🚀 快速开始

### 本地运行

**方法 1: 完整运行（前端 + 后端）**

1. 启动后端 API 服务器

```bash
cd server
npm install
npm start
```

后端 API 将运行在 `http://localhost:3000`

2. 启动前端服务器（新开一个终端）

```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 或使用 Node.js
npx http-server
```

3. 访问 `http://localhost:8000`

4. **使用定位功能**：
   - 点击地图右下角的定位按钮（📍）
   - 允许浏览器访问您的位置
   - 查看附近的景点和距离信息

5. **使用距离筛选**：
   - 在侧边栏"我的位置"区域，选择距离范围（1km/5km/10km/50km）
   - 地图会自动筛选显示范围内的热点
   - 点击"按距离排序"按钮，按距离从近到远排序

**方法 2: 仅前端运行（使用模拟数据）**

1. 直接用浏览器打开 `index.html`

```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 或使用 Node.js
npx http-server
```

2. 访问 `http://localhost:8000`

注意：如果后端服务器未运行，前端会自动降级使用模拟数据。

## 🎯 使用指南

### 基本操作

1. **浏览地图** - 拖拽、缩放地图查看热点
2. **查看详情** - 点击热点标记查看详细信息
3. **筛选热点** - 使用侧边栏筛选器过滤热点
4. **切换地图** - 在中国地图和世界地图之间切换

### 快捷筛选

- 📍 **全部热点** - 显示所有类型的热点
- 🍜 **美食探店** - 美食相关热点
- 🏝 **旅游打卡** - 旅游相关热点
- 🎪 **漫展活动** - 活动和 ACG 热点
- 📰 **社交热点** - 社交新闻热点

### 用户位置定位

1. **点击定位按钮**：地图右下角的定位图标（📍）
2. **授权位置**：允许浏览器访问您的位置
3. **查看位置**：地图上会显示您的位置标记（蓝色圆点）
4. **距离计算**：系统会自动计算所有热点到您的距离
5. **查看详情**：点击热点可查看距离信息和导航按钮

### 距离筛选

在侧边栏"我的位置"区域，选择距离范围：
- **全部**：显示所有热点（不筛选）
- **1km**：显示 1 公里内的热点
- **5km**：显示 5 公里内的热点
- **10km**：显示 10 公里内的热点
- **50km**：显示 50 公里内的热点

### 距离排序

点击"按距离排序"按钮：
- **首次点击**：按距离从近到远排序
- **再次点击**：恢复默认排序（按影响力）

### 距离统计

定位成功后，统计面板会显示：
- 📏 **最近距离**：最近的热点距离
- 📏 **平均距离**：所有热点的平均距离
- 📍 **5km 内**：5 公里内的热点数量
- 📍 **10km 内**：10 公里内的热点数量
- 📍 **50km 内**：50 公里内的热点数量

### 平台筛选

勾选或取消勾选各平台复选框，筛选指定平台的热点：
- 📕 小红书
- 📟 微博
- 🎵 抖音
- 📺️ B站

## 🔧 配置说明

### API 端点

后端 API 提供以下端点：

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | `/api/health` | 健康检查 |
| GET | `/api/hotspots` | 获取热点列表（支持筛选） |
| GET | `/api/hotspots/:id` | 获取热点详情 |
| GET | `/api/hotspots/search` | 搜索热点 |
| GET | `/api/stats` | 获取统计数据 |
| POST | `/api/fetch` | 触发数据抓取 |
| POST | `/api/hotspots` | 添加新热点 |
| PUT | `/api/hotspots/:id` | 更新热点 |
| DELETE | `/api/hotspots/:id` | 删除热点 |

### 修改热点数据

编辑 `js/data/hotspots.js` 文件，修改或添加热点数据。

### 修改地图中心点

编辑 `js/app.js` 中的 `initMap()` 函数：

```javascript
map.setView([纬度, 经度], 缩放级别);
```

### 修改影响力评分算法

编辑 `js/utils/influence.js` 文件中的评分函数。
